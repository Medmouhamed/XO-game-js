# ❌ Modern XO (Tic-Tac-Toe) Suite ⭕

A feature-rich Tic-Tac-Toe game environment featuring multiple game modes and difficulty levels, built with a sleek, modern UI.

## 🎮 Game Modes
* **Player vs Player (PVP)**: Local multiplayer mode to play with a friend.
* **Player vs Computer (PVC)**: Challenge the AI with three distinct difficulty settings:
    * **Easy**: A random-move  for beginners.
    * **Middle**: looks for immediate winning or blocking opportunities.
    * **Impossible**:  utilizing tactical board analysis.

## 🛠️ Technologies Used
* **HTML5**: For the multi-page structure and game navigation.
* **CSS3**: Advanced styling including gradients, backdrop-filters, and interactive hover animations.
* **JavaScript (Vanilla)**:
    * **Algorithm Implementation**: Logical check for victory conditions.
   

## 📂 Project Structure
* `XO.html`: The main menu and game hub.
* `PVP.html`: Local 2-player game logic.
* `PVCeasy.html` / `PVCmiddle.html` / `PVCinposible.html`
* `XO.CSS`: Shared styles across all game modes.
